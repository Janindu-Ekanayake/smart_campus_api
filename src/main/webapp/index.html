Project Overview

The Smart Campus API is a RESTful web service developed using Java, JAX-RS, and Maven. It is designed to manage IoT infrastructure across a university campus. This application serves as the central data hub for monitoring campus facilities.

Key Capabilities:
  • Room Management: Create, retrieve, and delete campus rooms (e.g., lecture halls, labs) ensuring strict data integrity (rooms cannot be deleted if sensors are inside).
  • Sensor Tracking: Register and manage IoT sensors (Temperature, CO2, etc.) linked to specific rooms. Includes status monitoring (Active, Maintenance).
  • Data Telemetry: Log real-time readings from sensors to build a historical data stream of environmental conditions.
  • Robust Error Handling: Comprehensive global exception mappers returning clean, professional JSON errors (404, 422, 409, 403, 500) instead of Java stack traces.
  • HATEOAS Discovery: Auto-navigable API entry points providing dynamic links to related resources.
